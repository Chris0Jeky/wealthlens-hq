# North Stars

## Product Milestones

- First deployed chart page with cited data and downloadable CSV.
- Five permanent chart pages covering wealth, housing, tax, inheritance, and income/capital gains.
- First partner or external site embeds a WealthLens chart.
- First contributor lands a useful change.

## Quality Metrics

- Every chart has source URL, access date, methodology, and version date.
- Every chart is usable on mobile.
- Every chart meets WCAG AA minimum standards.
- Every data transformation is reproducible from scripts.

## Visibility Metrics

- GitHub stars and forks.
- Newsletter signups.
- LinkedIn/X/Bluesky saves, replies, and shares.
- Inbound messages from researchers, journalists, campaigners, educators, or volunteers.
