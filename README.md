# WealthLens HQ

WealthLens HQ is the command centre for WealthLens UK: strategy, research, tasks, outreach, contributor operations, automation, and planned software projects.

WealthLens UK is an open-source project making UK wealth inequality data visible, interactive, and impossible to ignore.

## What Lives Here

- `identity/` - personal and professional context for AI agents and collaborators
- `vision/` - mission, theory of change, milestones, and inspiration
- `strategy/` - career, brand, content, outreach, partnership, volunteer, funding, and growth playbooks
- `research/` - raw research, synthesised insights, data source registry, and reading list
- `tasks/` - inbox, active sprint, backlog, done log, outreach, social media, and learning tasks
- `projects/` - WealthLens dashboard, landing page, newsletter, and social media assets
- `automation/` - data pipelines, social media generation, research analysis, and workflows
- `people/` - contributor roles, onboarding, recognition, and community notes
- `legal/` - licences, governance notes, and privacy policy drafts
- `meetings/` - notes from calls and project discussions
- `journal/` - personal reflection and learning log

## Planned Dashboard Stack

- Backend: Python and FastAPI
- Frontend: Vue 3, TypeScript, Pinia, TailwindCSS
- Visualisation: D3.js or ECharts
- Data: reproducible pipelines in `automation/data-pipelines/`
- Standards: cited data sources with URL and access date, mobile-responsive charts, WCAG AA minimum accessibility

## Current Operating Rule

Start small. The first useful public product is not a giant dashboard; it is a small set of permanent, shareable, source-backed chart pages with downloads, methodology, and embed/export support.
