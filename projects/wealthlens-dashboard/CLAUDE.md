# Region: wealthlens-dashboard (frontend + backend)

The live public product — Vue 3 + TypeScript + Pinia + Tailwind frontend (GitHub Pages),
FastAPI backend. Root seam map: `/AGENT_MAP.md`.

## Invariants (data honesty is the product)

- Every public figure cites its source with URL + access date. Never change a public
  headline number without a cited source — that is a Chris decision, not an agent one.
- Provenance shown on Data Sources AND Methodology comes only from
  `frontend/src/constants/datasetProvenance.ts` (keyset guard test enforces both pages).
- WAS-derived charts carry the June-2025 accreditation-loss caveat
  (`research/methodology/was-caveats.md`), wired via aria-describedby.
- Charts are mobile-responsive and WCAG AA minimum; visual + aria share one data source.
- `frontend/public/data/*` is generated by the static-API script; `.gitignore` ignores it
  except a committed JSON whitelist. Regenerate via the script — do not hand-add new files
  there, and let a whitelist change be a deliberate `.gitignore` edit.
- New behaviour ships toggleable, default OFF.

## Verify

- Frontend: `make frontend-lint frontend-typecheck frontend-test` (+ `frontend-build`
  before merge; CI adds e2e + lighthouse).
- Backend: `make ci-quick` (backend lint + tests).
- Touched a user-visible page? Check accessibility in the same slice.
