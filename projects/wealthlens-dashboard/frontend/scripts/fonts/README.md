# OG Generator Fonts

These fonts are vendored so `npm run generate:og` is reproducible without
runtime network access.

Source packages:

- `Inter-Regular.ttf`: `@fontsource/inter@5.2.8`, `latin-400-normal.ttf`
- `Inter-Bold.ttf`: `@fontsource/inter@5.2.8`, `latin-700-normal.ttf`
- `PlayfairDisplay-Bold.ttf`: `@fontsource/playfair-display@5.2.8`, `latin-700-normal.ttf`
