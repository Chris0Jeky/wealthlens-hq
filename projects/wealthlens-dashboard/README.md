# WealthLens Dashboard

Planned public product for WealthLens UK.

## Goal

Publish source-backed, mobile-first, accessible chart pages that make UK wealth inequality data easier to inspect, share, download, and embed.

## Planned Stack

- Backend: Python and FastAPI
- Frontend: Vue 3, TypeScript, Pinia, TailwindCSS
- Visualisation: D3.js or ECharts
- Data: reproducible pipelines in `automation/data-pipelines/`

## Product Shape

Start with a small number of permanent chart pages. Each chart should include:

- Headline and short interpretation.
- Source URL and access date.
- Methodology note.
- Downloadable CSV.
- Version date.
- Mobile-responsive and WCAG AA accessible chart implementation.
- Embed/export path where practical.
