## Licence summaries

### OGL v3.0 (Open Government Licence)

- **Permits:** copy, publish, distribute, transmit, adapt, exploit commercially and non-commercially
- **Requires:** acknowledge the source
- **Full text:** https://www.nationalarchives.gov.uk/doc/open-government-licence/version/3/

### CC-BY 4.0 (Creative Commons Attribution)

- **Permits:** share, adapt, commercial use
- **Requires:** give appropriate credit, indicate changes, link to licence
- **Full text:** https://creativecommons.org/licenses/by/4.0/
