# .codex/ — WealthLens HQ Control Plane

This directory is the Codex workspace planning and execution system.

## Reading order

1. `memories/00_ACTIVE.md` — current focus areas and status board (read first, always)
2. `memories/program/00_READ_THIS_FIRST.md` — program-level context and index
3. `memories/decisions/` — captured design decisions (ADRs)
4. `memories/session_notes/` — running notes for multi-session work

## Conventions

- `00_ACTIVE.md` is the single source of truth for what is currently active.
- Decisions go in `decisions/` as `ADR-YYYYMMDD-short-name.md`.
- Session notes go in `session_notes/` with timestamps.
- Do not bulk-read archives or strategy docs from here unless the task requires them.
