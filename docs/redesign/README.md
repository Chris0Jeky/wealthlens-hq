# Frontend Redesign

This directory is the receiving area for design briefs, mockups, instructions, and resources for the WealthLens dashboard frontend redesign.

## How to use

Drop any of the following here:
- Design briefs and specifications (Markdown, PDF)
- Mockups and wireframes (images, Figma exports)
- Component inventories and style guides
- Reference sites and inspiration screenshots
- Technical requirements and constraints

## Planned stack

- **Framework:** Vue 3 + TypeScript
- **State:** Pinia
- **Styling:** TailwindCSS
- **Charts:** D3.js or ECharts (to replace current Plotly static HTML)
- **Build:** Vite
- **Testing:** Vitest + Vue Test Utils

## Current state

The frontend directory at `projects/wealthlens-dashboard/frontend/` is scaffolded and ready for `npm create vue@latest` or equivalent. The current v0.1 product is static Plotly HTML charts deployed to GitHub Pages.
