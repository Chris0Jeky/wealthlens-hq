# Code of Conduct

WealthLens HQ should be a respectful, practical space for people working with data,
software, research, and public-interest communication.

## Expected Behaviour

- Be respectful and assume good intent while staying clear about technical concerns.
- Give feedback on work, evidence, and decisions rather than on people.
- Welcome contributors with different levels of experience.
- Cite sources for factual claims and correct mistakes promptly.
- Keep discussions focused on improving the project and its public usefulness.

## Unacceptable Behaviour

- Harassment, intimidation, discrimination, or personal attacks.
- Sexualised language or imagery.
- Publishing private information without explicit permission.
- Deliberately introducing false data, hidden tracking, secrets, or unsafe code.
- Repeated disruption after maintainers ask for a change in behaviour.

## Reporting

If you see behaviour that violates this code, open a private channel with the
project maintainer rather than escalating in public. Include the relevant context,
links, and screenshots where possible.

Reports will be reviewed in good faith. Maintainers may remove comments, close
issues, block contributors, or take other proportionate action to protect the
project and its contributors.

## Scope

This code applies in repository issues, pull requests, discussions, project docs,
and related project communication spaces.
