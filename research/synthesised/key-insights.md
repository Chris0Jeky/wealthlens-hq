# Key Insights

This file holds the distilled 20% of research that should drive 80% of action.

## Current Synthesis

- WealthLens should start with small, shareable, permanent chart pages rather than a large general dashboard.
- Each public chart should have a source URL, access date, methodology note, downloadable CSV, and version date.
- The strongest public positioning is public-interest engineering: making unequal systems legible.
- The first product should be useful to journalists, campaigners, educators, researchers, and ordinary citizens without requiring them to trust uncited claims.
