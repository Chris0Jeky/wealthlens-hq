## Flagship projects

### WealthLens UK
Open-source project making UK wealth inequality data accessible, interactive, and impossible to ignore. Strategy, research, data pipelines, and planned interactive dashboard.

- **Stack:** Python, FastAPI, Vue 3, TypeScript, D3.js/ECharts, Pinia, TailwindCSS
- **Status:** Active (research and strategy phase, dashboard in planning)
- **Repo:** github.com/Chris0Jeky/wealthlens-hq

### Taskdeck
Local-first Kanban + capture pipeline for developers. Proposal-first automation: capture fast, review changes explicitly, preserve provenance, avoid silent mutations. 4,000+ commits.

- **Stack:** C#, .NET 8, Vue 3, TypeScript, SQLite, SignalR, Redis, Docker, Terraform
- **Status:** Active
- **Repo:** github.com/Chris0Jeky/Taskdeck

### Metric Algo / Metrix / Options (fintech platform)
Co-developed SaaS platform for options backtesting and strategy operationalisation. 25+ ORM models, JWT/RBAC, Redis job queue, WebSocket real-time updates, OpenTelemetry tracing, dual-database architecture.

- **Stack:** Python, FastAPI, React 18, TypeScript, MySQL/MariaDB, Redis, Docker
- **Status:** Active (pre-revenue, undergoing rebranding)
- **Repo:** Private
